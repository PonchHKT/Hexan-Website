# Hexan Website

This is a school project, make this website from a template example.<br>
I've used NextJS and Sass, implemented NetlifyCMS.<br>
Everything work well as the responsive, if you would like to check the demo :<br>
[Hexan](https://elastic-brown-5a88f2.netlify.app/)

## Installation

You need to have  <img alt="Node.js" width="15px" src="https://raw.githubusercontent.com/github/explore/80688e429a7d4ef2fca1e82350fe8e3517d3494d/topics/nodejs/nodejs.png" /> NodeJS latest version then run :

```bash
npm i -g
```

## How to run build

```bash
npm run dev
```
